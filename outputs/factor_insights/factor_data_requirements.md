# 询价前因子体系与待补数据清单

生成日期：2026-05-29

## 设计口径

正式模型只使用 T-6 询价前可获得信息。T-1/T+1 字段只能作为研究对照，不进入正式模型。

因子构建借鉴量化因子投资的思路：每个因子都应有稳定的数据来源、明确的经济含义、可解释的方向，并能按板块和时期做分层验证。

## 可纳入模型的因子

| 因子组 | 因子 | 选用理由 | 当前状态 | 需要补的数据 |
|---|---|---|---|---|
| 市场流动性 | 全市场成交额 `market_turnover_ma20` | 成交额高代表资金活跃，网下申购更容易拥挤；当前 SHAP 贡献排名靠前 | 已有 | 可继续补日成交量、换手率 |
| 市场流动性 | 成交额一年分位 `market_turnover_pct_rank_1y` | 用分位数消除长期成交额中枢变化，便于跨年份比较 | 已有派生 | 无 |
| 市场流动性 | 成交额短长均线比 `market_turnover_ma20_over_ma60` | 衡量短期资金突然升温，而不是绝对成交额水平 | 已有派生 | 无 |
| 板块流动性 | 主板/创业板/科创板/北交所成交额 | 行业或板块热度可能不同，全市场成交额无法解释板块分化 | 缺失 | 各板块日成交额、成交量、换手率 |
| 行业景气度 | 申万一级行业成交额/成交量 | 如果某行业被高频交易或景气度高，该行业新股打新热度可能更高 | 缺失 | IPO 所属申万一级行业、行业日成交额、行业换手率 |
| 行业景气度 | 申万行业收益率/动量 | 行业上涨代表景气度和资金关注度提升 | 缺失 | 申万一级行业指数日收益率 |
| 行业景气度 | 行业成交额分位 | 用分位衡量行业是否处于历史热度高位 | 缺失 | 行业成交额历史序列 |
| 新股赚钱效应 | 近20只 IPO 首日涨幅 | 新股赚钱效应强，机构参与意愿通常更强 | 已有 | 可细化为同板块、同行业近20只 |
| 新股风险偏好 | 同板块近10只破发率 | 破发率高会打击打新热情，降低申购拥挤度 | 已有 | 可增加同行业破发率 |
| IPO 供给拥挤 | 同期 IPO 数量 | 同一窗口新股多会分流资金，也可能代表发行热窗口 | 已有 | 可补未来14/30天发行日历 |
| IPO 供给拥挤 | 同期网下发行量合计 | 比单纯数量更能衡量资金需求总量 | 已有派生 | 可补预计募资额合计 |
| 发行供给 | 回拨前网下发行量 | 供给越大，理论中签率越高 | 已有 | 无 |
| 发行供给 | 网下发行占比 | 反映网下投资者可分配盘子大小 | 已有 | 无 |
| 战略配售 | 战略配售占比 | 战配占比高可能压缩网下可分配量，也可能代表项目质量较高 | 已有 | 战投类型、跟投主体 |
| 申购规则 | 申购上限/下限/步长 | 直接影响机构可申购规模和报价离散度，当前贡献高 | 已有 | 市值门槛、A/B/C类配售规则 |
| 估值 | 行业 PE、可比公司 PE | 估值环境影响询价前吸引力 | 已有部分 | 发行估值区间、预计募资额 |
| 公司质量 | ROE、毛利率、净利率、营收增速 | 对应量化里的 Quality，基本面好通常更受机构追捧 | 缺失 | 招股书财务指标、Wind 财务摘要 |
| 成长性 | 营收 CAGR、利润 CAGR、研发费用率 | 高成长/高研发可能提升机构关注度，尤其科创板/创业板 | 缺失 | 近3年财务指标、研发费用 |
| 承销商声誉 | 主承销商/保荐机构 | 头部券商项目可能更容易获得机构关注，也代表定价和销售能力 | 缺失 | 主承销商、保荐机构、历史项目表现 |
| 行业/主题 | 是否热门赛道 | 热门赛道新股更容易拥挤 | 缺失 | 行业标签、概念标签、主题分类 |

## 下一步数据补充优先级

1. **板块和申万一级行业成交额/成交量**
   - 最贴合当前想法，能解释“某个板块、某个行业近期特别热”。
   - 建议先拿日频数据，派生 20日均值、60日均值、20/60 比值、一年分位。

2. **IPO 所属申万一级行业**
   - 这是行业景气度因子的连接键。
   - 没有行业映射，就无法把行业成交额接到每只新股。

3. **公司质量和成长性财务指标**
   - 用来构建 Quality/Growth 因子。
   - 至少补 ROE、毛利率、净利率、营收增速、利润增速、研发费用率、资产负债率。

4. **主承销商/保荐机构**
   - 用来分析机构销售能力和项目声誉。
   - 可以派生承销商历史 IPO 平均首日涨幅、破发率、平均超额认购倍数。

5. **市值门槛和配售规则**
   - 直接影响参与者结构。
   - 对解释网下投资者行为很重要。
